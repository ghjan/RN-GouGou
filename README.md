# RN-GouGou
一款配音短视频app，目前只开发了ios版本,
## 技术栈:RN+Node+es6+mongodb

#项目启动
##客户端
* 安装RN开发环境
* npm install 
* npm install rnpm -g  rnpm 
* react-native run-ios

## 服务端
* yarn
* yarn run dev

# 功能介绍
### 列表首页
* 下拉更新
* 上拉加载更多
* 点赞
* 点击视频进入详情
